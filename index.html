ABOUT

I am currently a Research and Teaching Assistant at TOBB University of Economics and Technology. I am broadly interested in computer architecture. I worked on memory (DRAM and hybrid memory systems), and hardware software codesigns, and I am currently interested in hardware security and machine learning solutions for computer architecture problems. Most recently, my paper "DR-STRANGE: End-to-End System Design for DRAM True Random Number Generators" got accepted to HPCA 2022.

I joined Kasirga Microprocessors' Lab in 2016 during my first year as a Bachelor's student at TOBB ETU. Since then, I had the chance to work on many exciting projects including two RISC-V processor design projects using Chisel3 and Verilog. Besides the hardware design projects, I worked on system- and memory-level simulation using system and DRAM simulators for an energy-efficient DRAM refresh mechanism project.

I joined Safari Research Group led by Prof. Onur Mutlu at ETH Zurich in 2018. During my internships, I worked (i) on a hardware design project focused on cross-layer communication, and (ii) on software tools to enable accurate and fast system-level simulation. I contributed to a metadata management infrastructure project that has a RISC-V prototype. I built a CPU and memory trace generator tool based on Intel Pin to alleviate simulation overheads and created data dependency and cache models to increase simulation accuracy. I also worked on extending Ramulator, a cycle accurate DRAM simulator, with emerging memory technologies such as STT-RAM and PCM enabling accurate simulations for these devices.
